# MOMO

Version: MVP 0.1

## Overview
MOMO is a mobile application for Korean mothers.
Goal: help moms create and discover playdates.

## Core Principle
Everything is a Card.

Card Types:
1. Playdate Card
2. Post Card

## MVP Screens
- Home Feed
- Create Selection
- Create Playdate
- Create Post
- Playdate Detail
- Post Detail
- Profile

## Navigation
- Home
- Create
- Profile

## Rules
- Use mock data first.
- No backend initially.
- Do not over-engineer.
- Every feature must help moms connect offline.
